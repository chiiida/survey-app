# Survey App

## Project setup
---
### Prerequisites
- Xcode
- CocoaPods
- Xcode Command Line Tools and Fastlane

### Install Pods
Install the latest pods:
```
$ pod install
```

## Test
---
To run the test using Fastlane, run this command:
```
$ fastlane ios allTests
```
